## 📖 Overview

This Docker container provides a **ready-to-use VPN server** with both OpenVPN and Ocserv (Cisco AnyConnect) protocols. Perfect for secure remote access to your network.

| Feature | Details |
|---------|---------|
| **Base Image** | Ubuntu 24.04 LTS |
| **VPN Protocols** | OpenVPN + Ocserv (Cisco AnyConnect compatible) |
| **Size** | 141 MB |
| **Architecture** | linux/amd64 |
